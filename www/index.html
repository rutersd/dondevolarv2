Acerca de
DroneCheck ES v1.0
¿Qué es esto?
DroneCheck ES es una herramienta de apoyo para pilotos de dron recreativos en España. Combina datos de espacios protegidos (OSM / Red Natura 2000) con una base de datos de ordenanzas municipales y normativa autonómica.
⚠ Aviso legal importante
Esta app es orientativa y no sustituye la consulta oficial.
Siempre debes verificar en ENAIRE antes de volar.
Los datos de ordenanzas y espacios protegidos pueden no estar actualizados. El desarrollador no se hace responsable de infracciones derivadas del uso de esta app.

Datos técnicos
Geocoding: Photon (Komoot/OSM) · Espacios protegidos: Overpass API (OSM) · Mapa: Leaflet + OpenStreetMap · Meteorología: Open-Meteo · Ordenanzas: base de datos manual + normativa oficial consultada.
Desarrollada con IA
Esta aplicación ha sido desarrollada íntegramente con la ayuda de Claude, el asistente de IA de Anthropic. Tanto el código como la base de datos de normativas han sido generados y revisados con asistencia de inteligencia artificial.

La información normativa ha sido contrastada con fuentes oficiales (ENAIRE, AESA, MITECO, Comunidades Autónomas) pero puede contener errores o estar desactualizada. Siempre verifica en las fuentes oficiales antes de volar.
