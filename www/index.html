¿Qué es esto?
DroneCheck ES es una herramienta para pilotos de dron recreativos en España. Combina datos de espacios protegidos (OSM / Red Natura 2000) con una base de datos colaborativa de ordenanzas municipales, generada automáticamente con IA y validada por la comunidad.

Datos técnicos
Geocoding: Photon (Komoot/OSM) · Espacios protegidos: Overpass API (OSM) · Ordenanzas: base local + generación automática vía Claude AI con búsqueda web · Los datos de ordenanzas son orientativos y deben verificarse con el ayuntamiento.
